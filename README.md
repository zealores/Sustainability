## Sustainability VS ESG
![ESG](https://cdn2.bigcommerce.com/server3300/hqacnni7/product_images/uploaded_images/sustainability-logo.png)

### Sustainability is a balance between to 3 major aspects (EES)
```bash
1- Environment
2- Economic
3- Social
```

**Sustainability** is not only beneficial for environment but the businesses themselves. Sutainability balances these three components by taking every aspect of it into account. Companies that adopt this strategy often gain a positive brand image and reduce cost through efficient energy cost. Beyond that sustainability cater both **Financial Performance (FP)** and **Non-Financial Performance (NFP)**. Financial Performance involve companies assets and revenues. Non-Financial Performance focuses on every other factors; Environment, Socials and Governance (ESG)

```
Sustainability = Financial Performance + Non-financial Performance
```

### Non-Financial Performance (ESG)
**Two Major Lenses of ESG** 
```
Organisational Behaviour
```
```
Business Models
````
Sustainability department only preview Organisational Behaviour. This explains behind every action of a company has an impact towards environment, social, and governance. These impact are measured by various methods. However ESG is not just about reporting, it's way beyond just showing data and amazing narrative. It is about coming up with a strategy that allow companies to keep improving in the long run.
